Welcome to The Ke3per

The Keeper or "The Ke3per" as it is named is a task or a to-do list project. It uses jQery, Javascript, PHP and XML to keep your tasks so you don't have to remember all of them. With the program you can create new tasks easily in the tasts page. When a task there has been finised simply tick the box at the left hand side of it and the task will be removed from the list and that task will be added to the page of finished tasks.
